# 🚀 Portfolio Website

A modern full-stack developer portfolio built with React, Express, MongoDB, and Tailwind CSS.

## ✨ Features

- Modern responsive UI
- Dynamic project showcase
- MongoDB database integration
- Admin authentication
- Resume download
- Contact section
- REST API backend

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- Lucide React

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt

## 📁 Project Structure

```
portfolio/
├── client/
└── server/
```

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
